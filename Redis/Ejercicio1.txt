2. LISTAS
Ejercicio: Cola de tareas
//1)Inserta varias tareas al inicio y al final de la lista.
lpush tareas "nacer"
lpush tareas "comer"
rpush tareas "morir"

//2)Muestra el número total de tareas.
llen tareas

//3)Elimina la primera tarea.
lpop tareas

//4)Elimina la última tarea.
rpop tareas

//5)Muestra todas las tareas restantes.
lrange tareas 0 -1

3. SETS (sin duplicados)
Ejercicio: Gestión de alumnos
//1)Añade varios alumnos a un conjunto.
sadd alumnos "Ana"
sadd alumnos "Alvaro"
sadd alumnos "Carlos"

//2)Intenta añadir un alumno repetido.
sadd alumnos "Alvaro"

//3)Muestra todos los alumnos.
smembers alumnos

//4)Comprueba si un alumno está en el conjunto.
sismember alumnos "Alvaro"

//5)Elimina un alumno.
srem alumnos "Ana"

//6)Crea un segundo conjunto con otros alumnos.
sadd alumnos2 "Hector"
sadd alumnos2 "Manu"
sadd alumnos2 "Anxo"
Obtén:
//6.1)los alumnos que están solo en el primero
srem alumnos

//6.2)los alumnos comunes a ambos conjuntos
sdiff alumnos alumnos2 // NO DA NINGUNO PORQUE NO HAY COINCIDENCIAS

4. ZSETS (ordenados)
Ejercicio: Ranking de jugadores
//1)Añade varios jugadores con puntuaciones.
zadd ranking 100 "Alvaro"
zadd ranking 200 "Hector"
zadd ranking 250 "Manu"

//2)Muestra el ranking de menor a mayor puntuación.
zrange ranking 0 -1

//3)Muestra el ranking de mayor a menor.
zrevrange ranking 0 -1

//4)Incrementa la puntuación de un jugador.
zincrby ranking 80 "Alvaro"

5. HASHES
Ejercicio: Perfil de usuario
//1)Crea un usuario con nombre, edad y puntos.
hset user:1 nombre "Alvaro"
hset user:1 edad 20
hset user:1 punto 120

//2)Obtén el nombre del usuario.
hget user:1 nombre

//3)Obtén todos los datos del usuario.
hgetall user:1

//4)Cuenta cuántos campos tiene el usuario.
hlen user:1

//5)Incrementa los puntos del usuario.
hincrby user:1 puntos
